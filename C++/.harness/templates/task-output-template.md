# Task Output Template

## 统一规则

1. 信息不足时，只输出“缺失信息清单”
2. 不允许夹带无关建议
3. 不允许跳过验证与合规摘要
4. 必须说明是否读取了 `.harness/error-journal.md`

## 模板 A：实施类任务

1. 需求理解
2. 关键信息
3. 变更内容
4. 验证结果
5. 历史错误对照检查
6. 合规检查摘要

## 模板 B：审查类任务

1. Findings
2. Open Questions
3. 风险摘要
4. 历史错误对照检查
5. 合规检查摘要

## 模板 C：模板 / 规范生成任务

1. 需求理解
2. 关键信息
3. 目录树
4. 文件正文
5. 验证矩阵
6. 合规检查摘要

## 模板 D：信息不足任务

1. 需求理解
2. 缺失信息清单
3. 风险说明
4. 合规检查摘要
