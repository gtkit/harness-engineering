# Project Scope

## 适用任务

1. 企业级 C++ 项目架构设计
2. Linux 进程、守护进程、系统服务治理
3. AI 模型推理接入、资源控制、失败退化
4. 构建系统、编译矩阵、静态检查、动态检查
5. 审查、回归、发布门槛与风险控制
6. 生成或维护 Harness 文件

## 默认假设

1. 使用 CMake
2. 目标平台为 Linux
3. 需要 GCC / Clang 双编译器矩阵
4. 需要 Debug / Release / Sanitizer
5. 需要结构化日志和可观测性
6. 需要回归与历史错误对照

## 非目标

1. 前端页面与 Web Demo
2. 与 C++ / Linux / AI 无关的泛化建议
3. 未经用户确认的技术选型结论
4. 无验证依据的上线判断

## 需用户补充项

1. AI Runtime
2. 测试框架
3. Benchmark 框架
4. 部署形态
5. 指标 / Trace 方案
6. 性能与资源预算
7. 安全合规基线
